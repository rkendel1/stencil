StencilGen

A professional multi-layer stencil generation tool for airbrush art. Upload any image, choose the number of layers, and generate clean vector stencil files suitable for cutting and sequential airbrushing to build up photorealistic images.

Pipeline: Enhanced preprocessing → B&W conversion → K-means/threshold/posterize segmentation → binary masking → validation & auto-fix → vectorization → optional AI airbrush compatibility evaluation → SVG/PDF/PNG export with registration marks.

AI Features: Groq Vision API analyzes layer sequencing, overspray protection, and airbrush build-up quality. Ensures each layer can be cut and airbrushed in sequence to recreate the original image. Perfect for t-shirts, canvas art, and custom paint work.
